# Bella Italia - Restaurant Website  

## 📌 Project Overview  
Bella Italia is a fully responsive and mobile-friendly restaurant website designed to showcase the restaurant's **menu, gallery, about page, contact details, and business information**. The project incorporates **modern web development principles**, ensuring a seamless and engaging user experience.

This website was built using:
- **HTML, CSS, JavaScript** for structure, styling, and interactivity
- **Bootstrap** for responsive design and layout efficiency
- **Owl Carousel & AOS** for animations and a dynamic image gallery
- **Google Maps API** for location integration

The project is hosted on **GitHub Pages** and can be accessed here:

## Source Files & Structure  
This project follows a modular structure to ensure scalability and maintainability:

- **HTML Pages**:
  - `home.html` - Home page
  - `menu.html` - Structured food menu
  - `about.html` - Restaurant history and details
  - `contact.html` - Contact form and Google Maps integration

- **CSS Stylesheets**:
  - `assets/css/style.css` - Main stylesheet for the site
  - `assets/css/responsive.css` - Mobile responsiveness and media queries

## 🏗 Implementation Details  

### **1. Header & Navigation**  
- Implemented a **fixed navigation bar** containing links to **Home, Menu, About, and Contact** pages.
- Ensured **cross-page consistency** by dynamically loading the navigation bar using JavaScript.

### **2. Hero Section**  
- Included a **high-quality banner image** with the restaurant’s name prominently displayed.
- Applied **CSS animations** for an engaging visual experience.

### **3. Menu Page**  
- Designed a **structured menu layout** using **cards** to display food items and prices in an organized manner.
- Utilized **flexbox and grid systems** to maintain readability and responsiveness.

### **4. Image Gallery**  
- Showcased **six high-quality images** of food and restaurant interiors.
- Integrated **Owl Carousel** to create a smooth, interactive slider effect.

### **5. About Section**  
- Developed a **concise yet informative history** of Bella Italia, highlighting its authenticity, special dishes, and mission.
- Ensured **readability and accessibility** by following proper text hierarchy.

### **6. Contact Page**  
- Embedded a **Google Map** to display the restaurant’s location.
- Included a **fully functional contact form** with fields for **Name, Email, and Message**.
- Implemented **form validation** to ensure proper input submission.

### **7. Footer Section**  
- Incorporated **business hours** and **social media links** (Facebook, Instagram, Yelp, TripAdvisor).
- Styled the footer to maintain a **consistent and professional aesthetic**.

---

## Mobile-First & Responsive Design  
- Utilized **CSS media queries** to adapt the site for various screen sizes (mobile, tablet, and desktop).  
- Implemented a **hamburger menu for mobile devices** to enhance user experience.

## Research & Design Inspiration  
During development, various restaurant websites were analyzed to understand industry trends, including:
- **Menu layout best practices**
- **Effective UI/UX design for readability**
- **Navigation simplicity & user flow improvement**

The design was uniquely crafted to reflect **Bella Italia’s branding and authenticity** while incorporating modern web design principles.
